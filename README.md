# 🏦 Taiwan Finance MCP Mega (台灣金融大聯盟) v1.2.0

這是一個基於 **Model Context Protocol (MCP)** 的重裝級金融數據伺服器，專為台灣市場開發。它整合了 **85 個真實金融工具**，數據直接串接政府官方及國際合法公開之 API，拒絕模擬數據。

## 💡 專案願景
將碎片化的台灣金融數據（證交所、央行、財政部、金管會等）統一封裝，讓 AI 模型能透過單一介面即時存取真實、活生生的台股、匯率、稅務、保險及企業財報。

## 🛠 核心工具類別 (85 Real Tools)

### 1. 台股市場 (TWSE/TPEx OpenAPI)
- **實時行情**：直接串接證交所 OpenAPI `STOCK_DAY_ALL`。
- **財務報表**：月營收 (`t187ap05_P`)、EPS 排名 (`t187ap14_L`)、股利分派 (`t187ap45_L`)。
- **籌碼與指數**：三大法人、融資融券、加權指數、櫃買指數。

### 2. 即時匯率 (Real ExchangeRate-API)
- **多幣別對台幣**：美金、日幣、歐元、人民幣、韓元等 10+ 幣別實時匯率。
- **自動更新**：數據每 10-60 分鐘自動同步真實市場匯率。

### 3. 加密貨幣 (CoinGecko API)
- **主流幣價**：BTC、ETH、SOL 等即時報價。
- **市場指標**：熱搜幣種、恐懼貪婪指數、全球市值統計。

### 4. 稅務、經濟與房地產 (Data.gov.tw)
- **經濟統計**：CPI 物價、GDP 成長、失業率、PMI 指數、貨幣供給 M2。
- **稅務與房產**：所得稅級距、囤房稅率、五大銀行房貸利率、實價登錄趨勢。

## 🛡️ 數據合法性說明
本專案嚴格遵守 Open Data 使用規範，數據來源包含但不限於：
- **台灣證券交易所 (TWSE) OpenAPI**
- **中央銀行 (CBC) 公開資料集**
- **財政部 (MOF) 稅務入口網**
- **行政院主計總處 (DGBAS)**
- **公開資訊觀測站 (MOPS)**

- **Command**: `python` (或指向虛擬環境中的 python 路徑)
- **Args**: `src/taiwan_finance_mcp_mega/server.py`
