# 🏦 Taiwan Finance MCP Mega (台灣金融大聯盟)

這是一個基於 **Model Context Protocol (MCP)** 的重裝級金融數據伺服器，專為台灣市場開發。它整合了超過 **80 個專業金融工具**，數據來源皆為政府官方及合法公開之 API。

## 💡 專案願景
將碎片化的台灣金融數據（證交所、央行、財政部、金管會等）統一封裝，讓 AI 模型能透過單一介面即時存取台股、匯率、稅務、保險及企業財報，達成真正的「金融 AI 助理」。

## 🛠 核心工具類別 (80+ Tools)

### 1. 台股市場 (TWSE/TPEx)
- **個股行情**：上市櫃個股即時報價、成交量、五檔資訊。
- **盤勢分析**：加權指數、櫃買指數、今日漲跌幅排行、成交量排行。
- **籌碼數據**：三大法人買賣超、全市場融資融券餘額。
- **特殊交易**：盤中零股交易、盤後定價、鉅額交易摘要。

### 2. 匯率與大宗商品 (Forex & Commodities)
- **即時匯率**：美金、日幣、歐元、人民幣、韓元對台幣即時匯率。
- **銀行比價**：各大銀行牌告匯率即時比價、銀行同業拆款利率。
- **國際商品**：黃金現貨、WTI/布蘭特原油、銅價(領先指標)、天然氣。

### 3. 保險、基金與稅務 (Insurance, Funds & Tax)
- **保險精算**：壽險平均保費統計、理賠申訴率、保險公司資本適足率 (RBC)。
- **基金數據**：勞動基金(勞退/勞保)收益率、國內前五大 ETF 持股、共同基金淨值。
- **稅務法規**：綜合所得稅級距、遺贈稅免稅額、囤房稅率、土地增值稅公式。

### 4. 企業財報與政府數據 (Corporate & Government)
- **財務指標**：營收 YOY 成長排名、EPS 獲利排名、現金殖利率、本益比過濾。
- **政府採購**：決標總金額統計、十大得標商、科技類標案預算趨勢。
- **經濟指標**：CPI 物價、GDP 成長率、PMI 製造業指數、失業率統計。

## 🛡️ 數據合法性說明
本專案嚴格遵守 Open Data 使用規範，數據來源包含但不限於：
- **台灣證券交易所 (TWSE) OpenAPI**
- **中央銀行 (CBC) 公開資料集**
- **財政部 (MOF) 稅務入口網**
- **行政院主計總處 (DGBAS)**
- **公開資訊觀測站 (MOPS)**
